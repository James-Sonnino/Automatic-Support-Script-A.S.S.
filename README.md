# Automatic Support Script (A.S.S)

The ASS will extract basic information on the customer's server:

System check:
  - Gravity status
  - Htop
  - Df -h
  - lsblk
  - Pods that are currently down 
  - Product version
  - Nvidia-smi output
Logs:
  - syslog
  - logs and describe output of failed pods
